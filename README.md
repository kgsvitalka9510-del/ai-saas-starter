## Features

- JWT Authentication
- Stripe Billing
- AI Integration (OpenAI)
- Responsive Dashboard
- Usage Analytics

## Quick Start

```bash
git clone https://github.com/kgsvitalka9510-del/ai-saas-starter.git
cd ai-saas-starter
pip install -r requirements.txt
uvicorn main:app --reload
```
